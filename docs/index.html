Qual o nosso objetivo.

Este projeto visa a criação de um sistema digital que gere os horários automaticamente, facilitando a administração dos agendamentos.

Nosso Processo.

Este sistema inclui um programa desktop que funciona de forma hierárquica, onde os administradores terão total controle e acesso, enquanto os dentistas comuns não terão acesso a algumas funcionalidades, tais como alteração de login e senha.

Nossos Motivos.

Após algumas pesquisas em consultórios, foram percebidas diversas defasagens no gerenciamento, tais como um grande volume de cadastros, formulários e registros, e consequentemente, uma enorme dificuldade no controle de informação, bem como grande risco de perda de dados.

Nossa Missão.

O sistema SmilIT foi planejado como uma solução ao problema da ineficácia dessas gestões. Este Software tem como intuito realizar o cadastro de pacientes, dentistas e secretários de uma forma mais prática, eficiente e segura do que a forma manual, entretanto este Software também permitirá que as fichas sejam impressas.

Funcionalidades de nosso programa.

Gerenciamento de Pacientes.

Campo onde é permitido o cadastro, a alteração, a exclusão e a impressão dos registros dos pacientes.

Gerenciamento de Avaliações.

Campo onde é permitido o cadastro, a alteração e a exclusão das fichas de avalição dos pacientes.

Gerenciamento de Dentistas.

Campo onde é permitido o cadastro, a alteração e a exclusão dos registros dos dentistas.

Gerenciamento de Tratamentos.

Campo onde é permitido o cadastro, a alteração e a exclusão dos registros dos tratamentos dos pacientes.

Gerenciamento de Usuários.

Campo onde é permitido o cadastro, a alteração e a exclusão dos registros dos usuários.

Gerenciamento de Agendamentos.

Campo onde é permitido o cadastro, a alteração e a exclusão dos registros de horários agendados.
